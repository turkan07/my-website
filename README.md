## ğŸ“ Customization Guide

### 1. **Update Personal Information**
Edit the HTML files to replace placeholder text with your information:
- Change "Turkan Mirzaliyeva" to your name
- Update "18" to your age
- Modify university and school names
- Update email addresses
- Change hobbies and skills

### 2. **Replace Images**
Save your actual images in the `images/` folder:
- `profile.jpg` - Your profile photo (200x200px recommended)
- `project1.jpg` - Project 1 image (320x220px recommended)
- `project2.jpg` - Project 2 image (320x220px recommended)
- `project3.jpg` - Project 3 image (320x220px recommended)

Update the `src` attributes in HTML files to match your image names:
```html
<img src="images/profile.jpg" alt="Your Name">
```

### 3. **Update Project Information**
Edit the project detail pages (`projects/project*.html`):
- Change project titles
- Update descriptions and content
- Add actual project links and videos
- Modify technologies and skills used

### 4. **Customize Colors (Optional)**
Edit `css/style.css` and change the CSS variables in `:root`:
```css
:root {
    --primary-dark: #0f172a;
    --accent-blue: #3b82f6;
    /* ... update other colors ... */
}
```
